# Proje Özeti

Google Sheets Altyapılı Kişisel Finans Web Arayüzü

## Genel Bakış
Kullanıcının kişisel bütçesini ve harcamalarını takip ettiği Google Sheets tablosuna mobil uyumlu, hızlı ve modern bir arayüz üzerinden veri girmesini sağlayan web uygulaması.

## Temel Hedefler
- **Mobil Kolaylığı**: verileri telefondan hızlıca girmek.
- **Modern Arayüz**: Karanlık mod, minimalist tasarım.
- **Entegrasyon**: Google Sheets API ile gerçek zamanlı veri yazma.
- **Kategorizasyon**: Settings tablosundan dinamik kategori (Gelir/Gider) çekme.
- **Dağıtım**: Coolify (Docker) üzerinde çalışabilirlik.
